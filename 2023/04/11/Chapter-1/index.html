快速掌握 Vue3 基础语法

2023-04-11

Vue2 与 Vue3 的对比

  • Vue2 对 TypeScript 支持不友好，所有属性都放在 this 对象上。
  • Vue2 架构层面对跨平台 DOM 渲染开发支持不友好。
  • Vue3 的 Template 支持多个根标签，Vue2 不支持。
  • Vue3 对虚拟 DOM 进行了重写、对模板的编译进行了优化。
  • …

setup() 函数

setup()是 Vue3 专门为组件提供的新属性。 setup() 在 beforeCreate() 之后和 created() 之前执行，
Vue3 取消了这两个 2 版本的生命周期钩子函数，统一用 setup() 代替，该函数相当于一个生命周期函数，
Vue2 中的 data，methods，等全部都用对应的新增 API 写在 setup() 中。

  • props 用来接收 props 数据。
  • context 用来定义上下文，上下文对象中包含了一些属性，这些属性在 Vue2 中需要通过 this 才能访问到，但在 setup() 中无法访问到 this
  • return {} 返回值，返回的是响应式数据。

reactive() 函数

reactive() 创建响应式数据对象，在 setup() 中 return 出去，即可在 template 中使用。

ref() 函数

ref() 用来创建一个响应式的数据对象，ref() 返回值是一个对象，这个对象上只包含一个 value 属性，
只在setup()函数内部获取 ref() 对象的值需要加上 .value。

在 reactive() 对象中访问 ref() 创建的响应式数据对象时，不需要加 .value。

isRef() 函数

isRef() 用来判断某个值是否为 ref() 创建出来的对象。

toRefs() 函数

toRefs() 可以将 reactive() 创建出来的响应式对象，转换为普通的对象，这个普通对象上的每个属性节点，都是 ref() 类型的响应式数据。

computed() 函数

computed() 用来创建计算属性，和前面一样，它返回的值是一个 ref() 对象。里面可以传方法，或者对象。
